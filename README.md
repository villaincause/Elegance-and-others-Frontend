# 🛒 Elegance & Others — Frontend

This is the standalone frontend repository for the **Elegance & Others** e-commerce platform. Designed with a "high-end minimalist" aesthetic, this project emphasizes clean UI, fast performance, and a seamless shopping experience.

## 🚀 Tech Stack
- **Framework:** [React](https://reactjs.org/) (initialized via [Vite](https://vitejs.dev/))
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** Context API / Redux Toolkit
- **Icons:** [Lucide React](https://lucide.dev/)
- **API Client:** [Axios](https://axios-http.com/)

## ✨ Key Features
- **Minimalist Design:** A premium look and feel with a focus on typography and whitespace.
- **Lightning Fast:** Optimized for speed using Vite's build toolchain.
- **Responsive UI:** Fully compatible with mobile, tablet, and desktop views.
- **Real-time Integration:** Connected to a Node.js/Express backend for product management and user sessions.
